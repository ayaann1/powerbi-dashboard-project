## Overview
This project presents a Power BI dashboard created to analyze a Data Professional Survey.
The dashboard provides insights into demographics, job roles, salaries, programming
languages, work-life balance, and career difficulty in the data field.

## Dashboard Insights
The dashboard highlights:
- Total number of survey participants
- Average age of respondents
- Average salary by job title
- Popular programming languages among data professionals
- Country-wise participation
- Satisfaction with salary and work-life balance
- Difficulty level of breaking into the data profession

## Tools Used
- Power BI Desktop
- Microsoft Excel
